# hypesquad
get your hypesquad back! discord removed unfort :[

A simple Python script that allows you to request and join a HypeSquad house via Discord's remaining API endpoint.

⚙️ How to use:
1. Open script.py in Visual Studio Code, Notepad++, or any text editor.
2. Replace "YOUR_TOKEN_HERE" with your account token. Keep this file local and private—never share your token with anyone.
3. Choose your house ID (1 = Bravery, 2 = Brilliance, 3 = Balance).
4. Run the script.

Working as of 06/20/2026.
